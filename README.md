# Prework Study Guide Webpage

## Description

Provide a short description explaining the what, why, and how of your project. Use the following questions as a guide:

- What was your motivation?
- Why did you build this project? (Note: the answer is not "Because it was a homework assignment.")
- What problem does it solve?
- What did you learn?

## Usage

The prework Study Guide website contains four fundamental sections, with detailed notes, to get you started with your software developing career i.e. HTML, CSS, JavaScript, and Git. If you open the DevTools console, we can see the JavaScript running. It will list the topics we learned from prework and choose one topic for us to study first.
